## 프로젝트 목적
이 프로젝트는 스프링 웹플럭스와 몽고DB를 사용하여 실시간 채팅 기능을 구현한 간단한 프로그램 입니다.<br>
이를 통해 비동기 프로그래밍, NoSQL 데이터베이스 활용, RESTful API 에 대해 공부하기 위해 만들었습니다.

## 기술
* Java: JDK 8 </br>
* Spring WebFlux </br>
* Frontend: HTML/CSS/JavaScript</br>
* IDE: IntelliJ IDE</br>
* Database: MongoDB
